# :dart: Tools for Autodarts v2.2.3 Released! :dart:

### :sparkles: WLED Enhancements
- **Preset Mode**: Fetch and select presets directly from your WLED controller via dropdown
- **JSON API Mode**: Send custom JSON POST requests to WLED for advanced effect control
- **New Game Modes**: Added support for ATC, RTW, Shanghai, and Bob's 27 with `target[1-20,25,bull]` triggers
- **New Triggers**: `bulloff`, `gameshot_[player]`, `matchshot_[player]`, `board_stopped`, `calibration_started`, `calibration_finished`
- **Trigger Once Option**: Prevent duplicate effects when hitting the same field repeatedly

### :bug: Bug Fixes
- **WLED s25 Trigger**: Fixed Single 25 incorrectly triggering as `25` instead of `s25`
- **Gameshot Sounds**: Fixed gameshot/matchshot sounds getting triggered multiple times
- **Streaming Mode**: Fixed checkout suggestions not updating after darts are thrown
- **Zoom Feature**: Fixed board images not loading correctly
