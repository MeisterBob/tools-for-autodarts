# :dart: Tools for Autodarts v2.1.27 Released! :dart:

### :new: New Features
- **Quick Correction Scale Setting**: You can now adjust the size of the Quick Correction window with a scale slider (0.5x to 2.0x) in the settings
- **Sound FX Range Triggers**: Sound FX now supports point range triggers (e.g., `100-180` or `ambient_100-180`) similar to Caller and WLED features
  - Use range triggers to play sounds for any point value within a specified range
- **QR Code Tournament Support**: Enhanced QR Code feature with improved layout and styling for tournament pages
  - _Note: This feature is prepared for the upcoming local tournaments feature and will be accessible once Autodarts releases it_

### :sparkles: Enhancements
- **Quick Correction Positioning**: Improved Quick Correction window positioning
  - Window now automatically repositions itself to stay visible on screen
  - Better handling when the window doesn't fit in the original position
- **Sound FX Range Triggers**: Improved range trigger matching for more reliable sound playback
- **WLED Settings**: Added warning message for HTTP URLs to help prevent potential issues

### :bug: Bug Fixes
- **WLED Range Triggers**: Fixed crashes when using multiple range effects in WLED
- **Quick Correction Settings**: Fixed issue where Quick Correction settings modal wouldn't open when clicking on the feature card
